Doom Emacs

last updated 2021-07-03

Project-wide find and replace

  1. Do a project-wide search with SPC-/
  2. Change the search buffer into an editable one with C-c C-e
  3. This buffer can be edited like a normal one, so you can do a find and replace like %s/find/replace/g
  4. To reflect the edits back to all of the files, press Z Z
